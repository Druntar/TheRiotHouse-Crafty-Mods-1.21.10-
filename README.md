# TheRiotHouse-Crafty-Mods-1.21.10-
Zip file containing mods for My minecraft server.

# Installation
1. Follow the instructions [here](https://wiki.fabricmc.net/install) to install the latest Fabric Client for 1.21.10.
2. Download the [mods.zip](https://github.com/Druntar/TheRiotHouse-Crafty-Mods-1.21.10-/blob/main/mods.zip) file and extract it into the "mods" folder of your Minecraft installation.
